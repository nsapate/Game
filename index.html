Dodge Alert

A side-scrolling platformer built during college and cleaned up to run as a modern static browser game. Collect every gem, avoid the patrols, and survive the moving platforms.

What Changed

  • Responsive presentation around the fixed Phaser canvas
  • Modern HTTPS-safe styling with no external font dependency
  • Cleaner level teardown and moving-platform handling
  • DOM controls for start state and sound state

Project Notes

Built with Phaser and tilemap assets. The goal here is not a full rewrite, just a pragmatic refresh that keeps the original game playable and presentable on GitHub Pages.
